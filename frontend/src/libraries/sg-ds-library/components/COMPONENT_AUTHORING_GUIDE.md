# Component Authoring Guide

This folder follows the shared Workbench component authoring guide:

- `../../../../../../docs/WORKBENCH-V1-COMPONENT-AUTHORING-GUIDE.md`

Read that document before creating or editing components in this folder.

Local notes for this library:

- Component styles live in `sg-ds-library.css`.
- Component stories live beside the component source.
- Library exports are collected in `index.ts`.
- Library tokens live in `../tokens.json`.
- Generated project token CSS lives in `../../../workbench-tokens.css`.
- Keep matching files in `sg-ds-library-preset/components` synchronized when a
  component exists in both places.
